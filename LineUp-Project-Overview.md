# LineUp — Project Overview

See `LineUp-Project-Overview.docx` for the full formatted document.

## Quick Summary

LineUp is a two-sided marketplace connecting solo barbers with customers. Core features: smart scheduling, ASAP waitlist queue (SMS via Twilio), Stripe no-show protection, QR tag system, and barber storefronts.

**Stack:** Next.js · React Native · Node/Express · PostgreSQL · Prisma · Clerk · Twilio · Stripe · Vercel · Railway

**Repo:** https://github.com/sebastiangonzalezatl-spec/lineup-barber
